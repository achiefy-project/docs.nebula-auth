# Linux and macOS support

Currently, NebulaAuth officially supports Windows only.

This is because the application is built on WPF, which does not run on Unix systems such as Linux and macOS.

We would like to support these platforms in the future, but this currently requires significant work.

***

## macOS, unofficial method

Below is a way to run NebulaAuth on macOS through Crossover.

These instructions were provided by user _**O S**_. Thanks to them for sharing this method.

{% hint style="danger" %}
This method is unofficial and has not been tested by us.

You use it at your own risk.
{% endhint %}

### Instructions

1. Extract the archive with NebulaAuth on your Mac.
2. Install the [Crossover](https://www.codeweavers.com/crossover) application
3. Launch Crossover
4. In the top menu select:\
   **Bottle → New Bottle**
5. Select the **Windows 11** type and click **Create**
6. In the next window, find the message "Can't find what you're looking for? Install an unlisted application", click **Install an unlisted application**
7. When Crossover asks for the installation file, choose `NebulaAuth.exe`
8. Click the **Install** button

This section currently has no screenshots. If you can make them, please share them and we will add them to the documentation with attribution.
{% endhint %}
