# Linux and MacOS support

At the moment, NebulaAuth officially supports only Windows.

This is due to the fact that the application is built on WPF, which does not work on Unix systems, Linux and macOS.

In the future, we would like to add support for these platforms, but at the moment this requires significant effort.

***

## macOS, unofficial method

Below is a way to run NebulaAuth on macOS through Crossover.

The instructions were provided by user _**O S**_, thanks to them for this.

{% hint style="danger" %}
This method is not official and has not been tested.

You use it at your own risk.
{% endhint %}

### Instructions

1. Extract the archive with NebulaAuth on your Mac.
2. Install the [Crossover](https://www.codeweavers.com/crossover) application
3. Launch Crossover
4. In the top menu select:\
   **Bottle → New Bottle**
5. Select the **Windows 11** type and click **Create**
6. In the next window, find the message "Can't find what you're looking for? Install an unlisted application", click **Install an unlisted application**
7. Next, it will ask you to provide the installation file, choose `NebulaAuth.exe`
8. Click the **Install** button

{% endhint %}
