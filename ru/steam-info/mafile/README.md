# Что такое мафайл

**maFile** — это файл, в котором хранится доступ к Steam Guard вашего аккаунта.

Он нужен для:

* генерации кодов входа
* подтверждения обменов и продаж
* управления аутентификатором

***

### Как NebulaAuth работает с maFile

При запуске NebulaAuth автоматически считывает все maFile из папки `maFiles`.

После этого они отображаются в разделе **«Аккаунты»** внутри интерфейса.

{% hint style="info" %}
В отличие от Steam Desktop Authenticator (SDA), NebulaAuth не требует файл `manifest.json`.
{% endhint %}

***

### Где находятся файлы

NebulaAuth использует несколько папок:

* `maFiles` — рабочие аккаунты
* `maFiles_removed` — удалённые аккаунты
* `maFiles_backup` — резервные копии

***

### Безопасность

{% hint style="danger" %}
Любой, у кого есть доступ к maFile, может использовать ваш Steam Guard.

Не передавайте эти файлы посторонним.
{% endhint %}

Рекомендуется:

* хранить maFile в безопасном месте
* делать резервные копии
* отдельно сохранять RCode (код восстановления)
