# Поддержка Linux и MacOS

На данный момент NebulaAuth официально поддерживает только Windows.

Это связано с тем, что приложение построено на WPF, который не работает на Unix-системах (Linux и macOS).

В будущем мы хотели бы добавить поддержку этих платформ, но на текущий момент это требует значительных усилий.

***

## macOS (неофициальный способ)

Ниже приведён способ запуска NebulaAuth на macOS через Crossover.

Инструкция предоставлена пользователем _**O S**_ — спасибо ему за это.

{% hint style="danger" %}
Этот способ не является официальным и не был протестирован.

Вы используете его на свой страх и риск.
{% endhint %}

### Инструкция

1. Распакуйте архив с NebulaAuth на вашем Mac.
2. Установите приложение [Crossover](https://www.codeweavers.com/crossover)
3. Запустите Crossover
4. В верхнем меню выберите:\
   **Бутылка → Новая бутылка**
5. Выберите тип **Windows 11** и нажмите **Создать**
6. В следующем окне найдите сообщение "Не можете найти то что ищете? Установите приложение, не внесенное в список" — нажмите **Установить приложение, не внесенное в список**
7. Далее вас попросит предоставить файл установки — выбираем `NebulaAuth.exe`
8. Нажмите кнопку **Установить**

{% hint style="info" %}
В данном разделе не хватает скриншотов, если у вас есть возможность их сделать — пожалуйста, поделитесь ими и мы добавим их в документацию с указанием авторства
{% endhint %}
