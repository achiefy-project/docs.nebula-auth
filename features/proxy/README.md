# Работа с прокси

NebulaAuth поддерживает использование прокси во всех аспектах работы с аккаунтами.

Прокси могут применяться:

* при авторизации
* при работе с Steam Guard
* при выполнении операций (обмены, подтверждения и т.д.)

Поддерживаются:

* **HTTP/HTTPS прокси**
* **SOCKS прокси**
* Прокси с авторизацией (с логином и паролем)
