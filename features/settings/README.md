#### Где сохраняются настройки

NebulaAuth хранит настройки в файле `settings.json` рядом с программой.

* файл сохраняется автоматически при изменении параметров
* если удалить `settings.json`, при следующем запуске будет создан новый файл с настройками по умолчанию
